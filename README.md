# Python Pattern Generator v1

A beginner Python pattern generator project using loops, nested loops, and menu-driven programming.

## Features
- Star Triangle
- Reverse Star Triangle
- Number Triangle
- Reverse Number Triangle
- Pyramid
- Inverted Pyramid
- Diamond
- Floyd's Triangle
- Alphabet Triangle
- Invalid Choice Handling

## Concepts Used
- Variables
- Input / Output
- if / elif / else
- for loops
- nested loops
- menu system
- ASCII characters using chr()
